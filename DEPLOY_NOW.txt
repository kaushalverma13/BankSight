⚠️  COMMON ISSUES & FIXES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "git is not installed"
   → Download from: https://git-scm.com/download/win
   → Run installer, click Next through everything
   → Restart PowerShell
   → Try again

❌ "fatal: not a git repository"
   → Make sure you're in d:\BankSight directory
   → Run: cd d:\BankSight
   → Try again: git init

❌ "ModuleNotFoundError" in Streamlit Cloud
   → Edit requirements.txt locally
   → Add missing module
   → Run: pip install -r requirements.txt
   → Push: git add requirements.txt
   → Push: git commit -m "Add missing module"
   → Push: git push
   → Streamlit redeploys automatically in 1-2 min

❌ Database error in Streamlit Cloud
   → This is normal first time!
   → Just click "Load Sample Data" in Settings
   → Database creates and persists automatically

❌ App runs slow / times out
   → Reduce sample data in app.py (line ~300)
   → Change: NUM_CUSTOMERS = 50 (was 100)
   → Push changes (git add . → git commit → git push)
   → Redeploys in 1-2 minutes

❌ "ModuleNotFoundError: psycopg2"
   → Already fixed! psycopg2-binary was removed from requirements.txt
   → Just push again: git commit --allow-empty -m "Trigger redeploy"
   → git push
