                        TROUBLESHOOTING QUICK FIX
                        ════════════════════════════

    ❌ Problem: "ModuleNotFoundError" on Streamlit Cloud
    ✅ Fix: Edit requirements.txt → Add module → Git push

    ❌ Problem: App loads slowly
    ✅ Fix: Reduce sample data size in app.py

    ❌ Problem: "git is not installed"
    ✅ Fix: Download from https://git-scm.com/download/win

    ❌ Problem: Database not found
    ✅ Fix: Click "Load Sample Data" in Settings (creates it)

    ❌ Problem: App 404 not found
    ✅ Fix: Wait 3-5 minutes for deployment to complete

    👉 For more: See DEPLOYMENT_CHECKLIST.md Troubleshooting section
